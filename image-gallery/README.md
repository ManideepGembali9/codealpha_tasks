Image Gallery 🖼️

This is a responsive image gallery website built using HTML, CSS, and JavaScript. It displays images in a clean grid layout with a modern user interface.

🚀 Features
- Responsive grid layout
- Smooth hover effects
- Lightbox / image preview (if added)
- Works on mobile and desktop

🛠️ Technologies Used
- HTML
- CSS
- JavaScript

 📂 Project Structure
image-gallery/
│── index.html
│── style.css
│── gallery.js
│── data.js

 ▶️ How to Use
1. Open the website in your browser
2. Browse through the image gallery
3. Click on images to view (if lightbox feature is included)
